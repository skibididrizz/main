# Overview

## Install

<Tabs>
<TabItem value="spec" label="In a spec" default>

```bash
npm install @skibididrizz/zod
```

</TabItem>
<TabItem value="library" label="In a library" default>

```bash
npm install --save-peer @skibididrizz/zod
```

</TabItem>
</Tabs>

Add the following line to `package.json`
```json
{
  "scripts":{
    "build": "tsp compile --emit @skibididrizz/zod ./lib/zod.tsp --option \"@skibididrizz/zod.emitter-output-dir={project-root}/src\""

  }
}
```
